# AI for Research Workshop

This is a teaching workspace for learning how to use AI coding agents for academic research.

## Environment

- Model: DeepSeek v4 Pro/Flash via Anthropic-compatible API
- Tool: Claude Code CLI

## Workshop Levels

1. **01-basics/** - CLI fundamentals: ask questions, read files, run commands
2. **02-mcp/** - Connect external tools (Semantic Scholar, ArXiv, Zotero)
3. **03-skills/** - Create reusable research workflows as slash commands
4. **04-harness/** - Multi-agent orchestration for complex research tasks

## Rules

- When searching for papers, always include DOI and year
- Prefer structured output (tables, bullet points) for literature reviews
- Always cite sources with author, year, and title
